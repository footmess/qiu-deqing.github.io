---
title: zsh基本操作
---

**ZSH**是一个强大的Unix shell提供很多强大功能, 比如:

- Drop-in replacement for BASH
- Command-line completion
- Shared history
- Spelling correction
- Theme-able
- Fully customisable
- Extended modules: TCP system, FTP, calendar functions and many more!

# centos 6安装zsh


1. 安装`epel-release`仓库

    ```
    yum install epel-release
    ```
2. 安装zsh

    ```
    yum install zsh
    ```
