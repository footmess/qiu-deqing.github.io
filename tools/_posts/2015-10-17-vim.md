## 文件操作

- `:q`退出, 如果未保存会提示保存
- `:w`保存修改
- `:x`保存并退出

## 常用设置

### 设置tab为2个空格

`vim ~/.vimrc`在用户目录下创建vim配置文件, 输入以下内容:

```
:set ts=2
:set expandtab
:$retab!
```

### 设置缩进为2个空格

`vim ~/.vimrc`添加如下配置

```
:set shiftwidth=2   // 设置缩进的空格数
:set autoindent     // 每行的缩进值与上一行相同
```

### 显示行号

`vim ~/.vimrc`添加配置

```
:set nu
```
