# 清除浏览器host缓存

地址栏访问: `chrome://net-internals/hsts#hsts`

点击右上角下拉三角-> tool下面clear cache


# 设置自定义搜索

设置 -> 搜索 -> 管理搜索引擎 -> 其他搜索引擎添加: 名字, 关键字, 内容:

    https://www.google.com/search?newwindow=1&q=%s+site%3Aqiudeqing.com

此时在地址栏输入关键词, tab就可以用指定文字执行搜索
