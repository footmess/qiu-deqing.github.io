## 添加行号

鼠标停靠在编辑器左边, 右键
![][1]

## 修改字体字号

快捷键`Cmd + ,`打开偏好: General -> Appearance -> Colors and Fonts -> java -> Java Editor Text Font, 然后点击右边Edit设置需要的字体.
![][2]

## 修改tab为4个空格

1. 快捷键`Cmd + ,`打开偏好: Java -> Code Style -> Formatter然后点击Edit
  ![][3]
2. 设置profile name, 在indentation选项卡下面选择spaces only, 保存即可
  ![][4]
